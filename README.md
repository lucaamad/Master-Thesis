# Master-Thesis
A collection of Python codes that reproduce the results of the paper "Prediction-Enhanced Monte Carlo: A Machine Learning View on Control Variate" by Fengpei Li, Haoxian Chen, Jiahe Lin, Arkin Gupta, Xiaowei Tan, Gang Xu, Yuriy Nevmyvaka. Agostino Capponi, Henry Lam. To reproduce the results of the paper, it is enough to download all the files and run the .py ones.
